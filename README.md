## Setup Instructions

1. Clone the repo
   git clone https://github.com/snehaxavier2/FortifyAI_Intergration.git
   cd FortifyAI_Intergration

2. Create virtual environment
   python -m venv venv
   venv\Scripts\activate

3. Install dependencies
   pip install -r requirements.txt

4. Download model weights (auto on first run)

5. Run server
   python manage.py runserver
